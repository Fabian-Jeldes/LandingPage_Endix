# Sitio Web de Arriaza & Ramírez Asociados

Este es un sitio web institucional y de aterrizaje (landing page) moderno y responsivo para el estudio jurídico de abogados **Arriaza & Ramírez Asociados**, especializado en Derecho Laboral.

El proyecto ha sido completamente rediseñado, optimizado y estructurado de forma modular utilizando tecnologías web estándar (HTML5, CSS3 y JavaScript moderno) para garantizar un excelente rendimiento, SEO óptimo y fácil mantenimiento.

## 📁 Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

*   **`index.html`**: Archivo principal que contiene la estructura semántica y el contenido de la landing page.
*   **`style.css`**: Hoja de estilos centralizada que define todo el sistema de diseño visual, variables de color (paleta HSL), tipografías, rejillas (grids), efectos de transición y animaciones interactivas.
*   **`script.js`**: Archivo de JavaScript que maneja toda la interactividad del sitio.
*   **`media/`**: Carpeta que almacena las fotos optimizadas de los abogados laborales y la administradora.
    *   La línea de tiempo interactiva paso a paso ("Tu caso, paso a paso").
    *   El sistema de modales dinámicos para los enlaces del pie de página (Términos, Privacidad, etc.).
    *   Efecto de revelación progresiva al hacer scroll (Scroll Reveal).
    *   Validaciones y comportamiento del formulario de contacto.

## 🚀 Características y Mejoras Implementadas

1.  **Diseño Visual Premium**: Se ha implementado un esquema de colores sofisticado utilizando tonos oscuros (Havana/Tinta) y detalles dorados, mejorando el profesionalismo y seriedad del sitio.
2.  **Sección de Equipo Optimizada**: Se alinearon los nombres y roles de los abogados de forma horizontal al lado de sus fotos (`[Foto] Nombre - Rol`), con esquinas redondeadas en las fotos (72x72px, 6px de radio), emulando un diseño de tarjetas moderno y elegante.
3.  **Tarjetas Interactivas**: Los cuatro puntos clave bajo la sección "Más que un abogado, un aliado" se han convertido en tarjetas de cuadrícula 2x2 con efectos dinámicos de elevación, sombras suaves y bordes inferiores animados (efecto hover).
4.  **Línea de Tiempo Interactiva**: La sección "Tu caso, paso a paso" permite hacer clic en cada paso, mostrando un indicador de progreso de carga suave antes de activar el paso correspondiente.
5.  **Sistema de Modales Moderno**: Los enlaces del pie de página (Términos y condiciones, Política de privacidad, Tratamiento de datos y Política de cookies) abren ventanas modales integradas con transiciones de escala y opacidad, evitando redirecciones externas y mejorando la retención de usuarios.
6.  **Optimización SEO y Accesibilidad**: Se utilizaron etiquetas HTML5 semánticas (`<header>`, `<section>`, `<footer>`), una jerarquía de títulos limpia (`<h1>` único) y atributos de accesibilidad adecuados.

## 🛠️ Cómo Visualizar el Proyecto

Para abrir y probar el proyecto de forma local:
1. Simplemente haz doble clic en el archivo `index.html` desde tu explorador de archivos para abrirlo en cualquier navegador web moderno.
2. Alternativamente, puedes usar una extensión como **Live Server** en Visual Studio Code para habilitar la recarga automática en tiempo real mientras editas.
